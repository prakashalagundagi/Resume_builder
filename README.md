# Resume Builder

A modern, responsive resume builder application built with React and Chakra UI. Create professional resumes in minutes with our intuitive interface.

## Features

- 🎨 **Modern UI** - Built with Chakra UI for a beautiful, responsive design
- 📝 **Real-time Preview** - See your resume update as you type
- 💾 **Download Support** - Export your resume as a text file
- 📱 **Mobile Friendly** - Works seamlessly on all devices
- ♿ **Accessible** - Built with accessibility in mind
- ⚡ **Fast** - Optimized for performance

## Sections

- **Personal Information** - Name, role, contact details, LinkedIn
- **Education** - Add multiple education entries with degree, institution, and CGPA
- **Skills** - Tag-based skill management
- **Work Experience** - Professional experience with descriptions
- **Projects** - Showcase your personal and professional projects

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/prakashalagundagi/Resume_builder.git
   cd Resume_builder
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
```

This builds the app for production to the `build` folder.

## Deployment

### GitHub Pages

The app is configured for automatic deployment to GitHub Pages. To deploy:

```bash
npm run deploy
```

This will build the app and deploy it to the `gh-pages` branch.
