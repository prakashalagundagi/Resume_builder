# ResumeGen

A modern, responsive resume builder application that enables users to create professional resumes quickly and efficiently. Built with React and Chakra UI, this application provides an intuitive interface for real-time resume editing and customization.

## ✨ Features

- **Live Resume Editor**: Real-time preview and editing capabilities
- **Professional Templates**: Multiple professionally designed resume templates
- **Export Functionality**: Download resumes as PDF or print directly
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices
- **Modern UI**: Clean, intuitive interface built with Chakra UI
- **Customizable Sections**: Add, edit, and remove resume sections as needed

## 🚀 Quick Start

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn package manager

### Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/resumegen.git
cd resumegen
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open your browser and navigate to `http://localhost:3000`

### Build for Production

```bash
npm run build
```

## �️ Tech Stack

- **Frontend**: React 17
- **UI Library**: Chakra UI
- **Styling**: Emotion (Chakra UI's styling engine)
- **PDF Generation**: jsPDF, html2canvas
- **Printing**: React-to-Print
- **Icons**: React Icons
- **File Handling**: React Files
- **Build Tool**: Create React App

## 📦 Dependencies

### Core Dependencies
- `@chakra-ui/react`: Modern UI component library
- `@emotion/react` & `@emotion/styled`: CSS-in-JS styling solution
- `react` & `react-dom`: Core React libraries
- `react-scripts`: Build and development scripts

### Feature Dependencies
- `html2canvas`: Convert HTML to canvas for PDF generation
- `jspdf`: PDF generation library
- `react-to-print`: Print functionality for resumes
- `react-files`: File upload and management
- `react-icons`: Icon library
- `uuid` & `uuidv4`: Unique identifier generation

## 🎯 Usage

1. **Create Resume**: Start with a blank template or choose from predefined templates
2. **Edit Content**: Modify personal information, work experience, education, and skills
3. **Customize Design**: Adjust colors, fonts, and layout to match your preferences
4. **Preview**: See real-time changes as you edit
5. **Export**: Download your resume as PDF or print directly
